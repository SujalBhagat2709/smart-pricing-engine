# Smart Pricing Engine

## Overview

A dynamic pricing engine that recommends prices based on:

- Sales
- Stock
- Competitor Pricing

---

## Files

- pricing_analyzer.py
- pricing_engine.py

---

## Run

```bash
python pricing_engine.py
```

---

## Example

Input:

```text
Product Name:
Gaming Mouse

Base Price:
1000

Current Stock:
20

Today's Sales:
50

Competitor Price:
1100
```

Output:

```text
Recommended Price:
₹1380.0
```
